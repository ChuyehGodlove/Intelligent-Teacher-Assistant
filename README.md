Intelligent Teacher AssistantAn AI-powered formative assessment platform designed to help teachers track student understanding in real-time. 
This system bridges the gap between traditional paper-based testing and modern data analytics using a hybrid "4+4" instructional model.
🚀 The "4+4" MethodologyThe platform is built around a specific instructional cycle:Instruction Phase: 4 hours of initial teaching.Diagnostic Check: 
A mid-topic evaluation using the platform to identify gaps before they compound.Refinement Phase: 4 hours of targeted instruction based on AI-generated insights.
✨ Key Features1. Dual-Path EvaluationTeachers can set both MCQ and structured questions based on their preferred evaluation methods.
Secure MCQs: Automatically graded by the system to provide immediate feedback on baseline comprehension.Structured Questions: 
Students can answer on tablets or scan and upload handwritten paper. At this level, only the teacher is allowed to mark these questions to ensure academic integrity.
2. AI-Powered "Meta-Analyst"After a teacher records marks and comments on structured papers, the AI performs a deep audit:Error Detection: 
Automatically flags questions marked "wrong" by the teacher for each student.Comment Categorization: 
Analyzes teacher feedback to distinguish between procedural errors (e.g., calculation mistakes) and fundamental conceptual gaps.Class-Wide Insights:
Identifies specific questions that more than half the class struggled with or failed.
3. Mastery Matrix & AnalyticsThe system aggregates all data into a "Class Understanding Index" ($CUI$):Performance Analysis: 
Tracks whether a student's score is above or below the class average to determine if they have truly understood the material.Interactive Student Portal:
A seamless interface for students to switch between MCQ and structured assessment sections.
🛠️ Technology StackFrontend: Next.js (App Router), Tailwind CSS, TypeScript.Backend: Python FastAPI.Database: PostgreSQL.AI/ML: 
OpenCV for image processing and Mathpix/NLP for handwriting and comment analysis.
